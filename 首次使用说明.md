# CampusPilot 启动器使用说明

本项目当前使用“启动器”方式运行，不是完整绿色版。

双击 `start_campuspilot.bat` 后，会自动完成以下操作：

1. 进入项目目录
2. 启动后端服务 `backend/app.py`
3. 自动打开浏览器访问 `http://127.0.0.1:5000/`

## 一、启动前必须准备的软件

请先在电脑上安装以下软件：

### 1. Python 3
建议安装 Python 3.10 及以上版本。

要求：
- 安装时勾选“Add Python to PATH”
- 安装完成后，在命令行执行 `python --version` 能正常显示版本

### 2. 项目依赖包
进入 `backend` 目录后执行：

```bash
pip install -r requirements.txt
```

当前依赖来自 `backend/requirements.txt`，主要包括：
- Flask
- flask-cors
- cryptography
- requests
- selenium
- beautifulsoup4

### 3. Git Bash
本项目包含大量 `.sh` 脚本，因此需要 bash 环境。

建议安装：
- Git for Windows（安装后自带 Git Bash）

安装完成后，系统应能使用 bash 相关命令。

### 4. Google Chrome
图书馆登录与 token 获取依赖 Chrome 浏览器。

### 5. ChromeDriver
Selenium 依赖 ChromeDriver 控制浏览器。

注意：
- ChromeDriver 版本要和本机 Chrome 主版本尽量对应
- 如果版本不匹配，自动登录/获取 token 可能失败

## 二、推荐目录结构

建议把启动器放在项目根目录，也就是和以下目录同级：

- `backend`
- `frontend`
- `shell`
- `database`
- `start_campuspilot.bat`

## 三、如何启动

### 先检查环境（推荐）
首次使用前，建议先双击：

- `check_environment.bat`

它会自动检查：
- Python
- Python 依赖包
- Git Bash
- Google Chrome
- ChromeDriver
- `frontend` / `shell` / `database` 目录是否存在

如果检查里出现错误，建议先修复后再启动项目。

### 方法 1：直接双击启动
双击项目根目录中的：

- `start_campuspilot.bat`

### 方法 2：命令行启动
也可以在项目根目录执行：

```bash
start_campuspilot.bat
```

## 四、启动器的 Python 选择规则

启动器会按下面顺序寻找 Python：

1. 优先使用项目根目录下的 `.venv\Scripts\python.exe`
2. 如果没有 `.venv`，则尝试使用系统 PATH 里的 `python`

所以推荐两种用法：

### 用法 A：使用系统 Python
直接安装 Python，并确保 `python` 命令可用。

### 用法 B：使用项目虚拟环境
在项目根目录创建 `.venv`，例如：

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r backend\requirements.txt
```

## 五、首次使用建议流程

1. 安装 Python、Git Bash、Chrome、ChromeDriver
2. 安装依赖包
3. 先运行 check_environment.bat 检查环境
4. 双击 start_campuspilot.bat
5. 浏览器打开 http://127.0.0.1:5000/
6. 登录系统
7. 绑定校园账号 / WebVPN
8. 进入图书馆相关页面完成一次登录
9. 再使用抢座、worker、课表等功能

## 六、常见问题

### 1. 双击后提示找不到 Python
说明本机没有安装 Python，或 Python 没有加入 PATH。

解决方法：
- 重新安装 Python，并勾选加入 PATH
- 或在项目根目录建立 `.venv`

### 2. 页面能打开，但启动抢座失败
常见原因：
- 没装 Git Bash / bash 环境
- WebVPN 未登录
- 图书馆 token 未获取到
- ChromeDriver 与 Chrome 版本不匹配

### 3. 浏览器没有自动打开
可以手动访问：

- [http://127.0.0.1:5000/](http://127.0.0.1:5000/)

### 4. Selenium / ChromeDriver 报错
请优先检查：
- Chrome 是否已安装
- ChromeDriver 是否存在
- ChromeDriver 版本是否与 Chrome 对应

## 七、如果后续要做 exe

当前建议先把 `start_campuspilot.bat` 转成“启动器 exe”。

注意：
- 这种 exe 只是简化启动方式
- 仍然需要用户提前安装 Python、依赖包、Git Bash、Chrome、ChromeDriver
- 它不是完整独立打包版
